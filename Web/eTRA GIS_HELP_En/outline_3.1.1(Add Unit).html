Add Unit

Only the first-level dispatcher can add the unit information. The hierarchical-level dispatcher has no permission.

The steps of adding the unit are as follows:

1) Click [Basis/Unit]. The Unit List pops up.

2) Click <Add>, and the “Add” dialog box pops up. Enter the unit information in the dialog box, shown in Figure 1.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Figure 1. Add the unit information

Note:

  • If the unit name already exists, it will fail to add the unit.

  • If the latitude and longitude already exists, or is beyond the latitude and longitude of the map, it will fail to add the unit.

  • If the uploaded file does not comply with the file format, or is 0KB, or exceeds the critical size, or the file suffix is modified exceptionally, it will fail to add the unit.

The steps of adding the subordinate unit are as follows:

1) Click [Basisn/Unit]. The Unit List pops up.

2) Click <Add sub-units> in the right of the unit, shown in Figure 2.

3) The “Add” dialog box pops up. Enter the sub-unit information in the dialog box.

3) Click <OK>.

Figure 2. Add the sub-unit
