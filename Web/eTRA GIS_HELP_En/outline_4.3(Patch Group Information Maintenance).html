Patch Group Information Maintenance

The patch group information parametrs are as follows:

Name Description  Type    Default

Group name

The name of the patch group

Character, 1~10 Chinese or English string

N/A

The unit

To which unit the patch group belongs

Character, 1~10 string

N/A

Member group

The groups associated to the patch group

Can select multiple groups, and the patch group contained GSSI is the valid GSSI in Group List or Multicast Group List

 

 

This chapter mainly describes:

  • Add Patch Group

  • Modify Patch Group

  • Delete Patch Group

  • View Patch Group

Note:

When the patch group information is created successfully, authorized dispatcher can initiate the “Patch group call”.
