Real-Time Trajectory

If the real-time trajectory function is enabled, the system will display the user’s trajectories in lines at real time. When the mouse is over a point, it will display the time of the point. If the real-time trajectory function is disabled, then the trajectories will disappear.

Here are some common methods of enabling/disabling the real-time trajectory function:

1) Right-click a mobile user on the map and select [Application service/Open real-time trajectory]. The “Select the trajectory style” dialog box pops up. Select the "Color" and "Width", and you can view the effect, shown in Figure 1.

Figure 1. Select the trajectory style

2) Click <OK>. The system will display the user’s movement trajectories in lines at real time. Right-click the user and select [Application service/Close real-time trajectory]. The trajectories will disappear.

3) Alternatively, click [Dispatch/Call panel] on the menu bar, and click "Real-time trajectory”. The “Select the trajectory style” dialog box pops up. Enter the terminal ID, or click <Select> to select the user, and then select the "Color" and "Width", shown in Figure 2.

Figure 2. Select the trajectory style

4) Click <OK>. The system will display the user’s movement trajectories in lines at real time. Right-click the user and select [Application service/Close real-time trajectory]. The trajectories will disappear.

Note:

If the user is in hidden status, open the real-time display first, then open the real-time trajectory.
