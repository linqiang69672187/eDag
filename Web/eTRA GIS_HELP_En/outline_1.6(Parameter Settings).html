Parameter Explanation

Name Description  Type  

Terminal timeout

How much time will the terminal be hidden on the map when it doesn’t report GPS

5/10/15/30/60 minutes
If the Hidden setting is not enabled, then no matter how long the timeout period of the terminal is set, it will not be hidden.

Hide

If the terminal didn’t report GPS within the set time, it will be hidden on the map

Check to enable the setting, uncheck to disable the setting

Map refresh time

Refresh the map according to the set time

Every 10/30/60 seconds

Map background image Select the background image to be uploaded png/jpg/bmp/jpeg/gif, not larger than 1MB
