BS Group Information Maintenance

The BS group information parametrs are as follows:

Name Description  Type    Default

BS group name

The name of the BS group

Character, 1~10 string

N/A

The unit

To which unit the BS group belongs

Character, 1~10 string

N/A

Member

The BS members associated to the BS group

Can select multiple BSs, and the BS group contains the BS name and BS number

 

 

This chapter mainly describes:

  • Add BS Group

  • Modify BS Group

  • Delete BS Group

  • View BS Group

Note:

When the BS group information is created successfully, authorized dispatcher can initiate the “BS group call”.
