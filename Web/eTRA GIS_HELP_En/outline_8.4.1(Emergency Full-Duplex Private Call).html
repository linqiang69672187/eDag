Emergency Full-Duplex Private Call

  • Call Panel: Initiate Emergency Full-Duplex Private Call; Receive Emergency Full-Duplex Private Call

  • Uplink Call Panel: Receive Emergency Full-Duplex Private Call; Initiate Emergency Full-Duplex Private Call

The steps of initiating an emergency full-duplex private call through call panel are as follows:

1) Click [Dispatch/Call panel] on the menu bar and click "Emergency call”. The “Emergency call” dialog box pops up.

2) Enter the terminal ID, such as 180008. The system will get the corresponding information automatically.

3) Select the “Call mode” as “Tel mode”, and click <Call>, shown in Figure 1.1.

Figure 1.1. The subscriber is ringing

4) After the user picks up, the call is set up, shown in Figure 1.2.

Figure 1.2. The subscriber has answered

5) Click <End> to end the call.

Note:

If you initiate an emergency call to a mobile user who has not registered in the system, the window will prompt, “The subscriber hangs up”.

The steps of receiving an emergency full-duplex private call through call panel are as follows:

1) If the dispatch console receives an emergency full-duplex call from the user 180008, the “Emergency call” dialog box is shown in Figure 1.3.

Figure 1.3. Call access request

2) Click <Answer>, and the call is set up, shown in Figure 1.4.

Figure 1.4. Call is set up

3) Click <End> to end the call.

The steps of receiving an emergency full-duplex private call through uplink call panel are as follows:

1) When the dispatch console receives an emergency full-duplex call from a user, such as user 180008, the private call panel is shown in Figure 2.1.

Figure 2.1. Call access request

2) Click <Answer>, and the call is set up, shown in Figure 2.2.

Figure 2.2. Call is set up

3) Click <Hang up> to end the call.

The steps of initiating an emergency full-duplex private call through uplink call panel are as follows:

1) Click <Call> in the private call panel to initiate the call, shown in Figure 2.3.

Figure 2.3. The subscriber is ringing

2) After the user picks up, the call is set up, shown in Figure 2.5.

Figure 2.5. The subscriber has answered

3) Click <Hang up> to end the call.

Note:

In the uplink call panel, you can only initiate an emergency full-duplex call to the terminal that already has the emergency full-duplex call records.
