Configuration User Information Maintenance

The configuration user name and ID is unique in the system. When we create a first-level unit, the system will automatically generate a first-level configuration user. The username is root and the password is 123456.

The configuration user information parametrs are as follows:

Name Description  Type    Default

The unit

To which unit the configuration user belongs

Character, 1~10 string

N/A

Username

The username of the configuration user

4~20 characters of letters, figures or underscores

N/A

Password

The password used to log in to the GIS system

Character, 1~13 string

N/A
