Modify Configuration User

The steps of modifying the configuration user are as follows:

1) Click [Basis/Config user]. The Configuration User List pops up.

2) Click <Modify> in the right of the configuration user, and the “Modify” dialog box pops up. Modify the configuration user information based on actual needs, shown as below.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Modify the configuration user information

Note:

  • The first-level configuration user can modify the configuration user information at all levels.

  • The hierarchical-level configuration user can only modify the configuration user information of his unit and sub-units.

  • Once a configuration user is created, the configuration user name cannot be modified.

  • The information of the logged configuration users cannot be modified.
