View Configuration User

The steps of viewing the configuration user are as follows:

1) Click [Basis/Config user]. The Configuration User List pops up.

2) Select the unit, or enter the configuration user name. Click <Search>. The list will display the configuration user information, shown in Figure 1.

Figure 1. Query the Configuration User List

3) Click the configuration user name to display the details, shown in Figure 2.
