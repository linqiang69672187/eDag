Watchlist

We can add certain mobile users to the watchlist, and the users will be marked with special icon.

Here are some common methods of configuring the watchlist:

1) Right-click a mobile user on the map and select [Application service/Add to watchlist]. A star icon will appear in the upper right corner of the user, shown in Figure 1. Right-click the user and select [Application service/Delete from watchlist]. The star icon will disappear.

Figure 1. Add to watchlist

2) Alternatively, search the user first, and click [Operate/Application service/Add to watchlist], shown in Figure 2. Click [Operate/Application service/Delete from watchlist] to delete the user from the watchlist.

Figure 2. Add to watchlist

3) You can also right-click on the map and select [Watchlist] to view the watchlist information and perform various operations, shown in Figure 3.

Figure 3. Watchlist
