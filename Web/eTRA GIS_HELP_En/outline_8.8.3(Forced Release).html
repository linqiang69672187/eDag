Forced Release

Forced release is set up based on discreet listening. When a monitored mobile user is in a call, the dispatcher can disconnect the call at any time, namely, the monitored party is forced to release the call.

  • Release Full-Duplex Call

  • Release Half-Duplex Call

For example, “180008” has been monitored, and is in a full-duplex call with “160001”. The steps are as follows:

1) Click <Forced release>. The full-duplex call ends, shown as below

2) At this time, the discreet listening also ends.

Forced release

For example, “180008” has been monitored, and is in a half-duplex call with “160001”. The steps are as follows:

1) Click <Forced release>. The half-duplex call ends.

2) At this time, the discreet listening also ends.

Note:

  • Forced release can only be realized in discreet listening status.

  • We cannot perform forced release after the break-in operation.
