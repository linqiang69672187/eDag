Unit Information Maintenance

There are two types of units: first-level unit and hierarchical-level unit. There is only one first-level unit, automatically created by the system and cannot be deleted.

The unit information parametrs are as follows:

Name Description  Type    Default

Unit name

The name of the unit

Character, 1~10 Chinese or English string (support Chinese & English & Figures mixing)

N/A

The higher-level unit

The higher-level unit of this unit

Character, 1~10 string

N/A

Latitude and longitude

Double-click anywhere on the map to obtain the latitude and longitude automatically

Longitude coordinates (0~180), latitude coordinates (0~90)

N/A

Icon

Upload icons to distinguish different units

png/jpg/bmp/jpeg/gif, not larger than 1MB

N/A

Remarks

Remark other information of the unit, such as address, phone, etc.

Character, not more than a string of 150 ASCII characters

N/A
