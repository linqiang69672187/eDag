Emergency Call

Emergency call is a call of the highest priority to preempt the resources. If an emergency call is initiated, the called partyâ€™s terminal will generate sound and light alarm, which will not be eliminated until the emergency call is released.

In the system, emergency call includes:

  • Emergency Full-Duplex Private Call

  • Emergency Half-Duplex Private Call

  • Emergency Group Call

The emergency call parameters are as follows:

ID

The ID of the terminal or group that initiated the call

Group name

The name of the group

Unit

To which unit the terminal or group belongs

Number

The serial number of the mobile user

Status

Call access request

The mobile user is calling the dispatch console

Call is being processed

The dispatch console calls out, waiting for the server to respond

The subscriber is ringing

The server has responded, waiting for reply

Call is set up

The user has replied, the server assigns the line resources and establishes the connection

Obtain authorization

The connection has been established, the voice permission has been obtained

PTT released

The user has released the PTT button, and the voice permission has been released

The subscriber hangs up

The connection is disconnected

Answer mode

Can be divided into auto-answer mode and non-auto-answer mode
