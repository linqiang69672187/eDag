Patch Group Call

The dispatch console can combine multiple talkgroups into one patch group. When the dispatcher initiates a patch call, all the talk group members in the patch group can receive the voices. If any member in a talk group uses group call, all talkgroup members in the patch group can receive the group call.

It contains three processes:

  • Patch: Select the talkgroups to be added to the patch group. That is, implement DGNA (Dynamic Group Number Assignment) on all talk group members in the patch group.

  • Patch call: a half-duplex call, consistent with the process of group call.

  • Cancel the patch: Cancel DGNA of all talkgroup members in the patch group.

Click [Dispatch/Call panel] on the menu bar and click “Patch group call". The “Patch group call” dialog box pops up, shown as below. We can perform related operations through the dialog box.
