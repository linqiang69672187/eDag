产品概述

基于eTRA的WEB GIS调度系统是东方通信股份有限公司根据客户的客观要求结合实际需求自主研发的地理位置信息调度系统，该系统主要服务于公安、消防、武警、政府等具有地面无线指挥调度需求的专网客户，满足众多行业应用中对GPS业务的实际需求，将无线指挥调度通信与GIS地理信息系统、GPS全球定位系统进行有效的结合，推出“点击指挥”及“图上作业”的全新指挥调度模式。

GIS调度系统主要通过IP网络与WEB服务器端相连，获取终端用户GPS位置信息，并在地图上进行实时位置显示；调度用户还可在界面上对终端用户进行调度呼叫操作。

GIS调度系统包括GPS解析服务器软件，调度呼叫控件及WEB GIS服务器。其中，GIS服务器用于解析eTRA系统发来的无线终端GPS位置信息以及实现GIS的相关处理；GIS调度台是用户进行指挥调度的操作终端，根据用户的使用要求可配置一台或多台。

GPS解析服务器软件安装

GPS解析服务器软件安装前，需确保运行环境满足如下的要求：

  1. 硬件环境：IBM兼容标准PC

  2. 软件环境：

  • 操作系统：Windows XP

  • eTRA API

  • 数据库：Microsoft SQL Server 2008

WEB GIS服务器安装

WEB GIS服务器安装前，需确保运行环境满足如下要求：

  1. 硬件环境：1CPU（2GHZ），1G RAM，160G HDD，DVD/CD光驱，10/100M自适应网卡*2

  2. 软件环境：

  • 操作系统：Windows Server 2003、Windows Server IIS6.0、.Net Framework 4.0

  • 数据库：Microsoft SQL Server 2008

  • FTP：Server-U

GIS客户端呼叫控件安装

调度客户端安装呼叫代理控件前，需确保运行环境满足如下要求：

  1. 硬件环境：1CPU（2GHZ），1G RAM，160G HDD，DVD/CD光驱，10/100M自适应网卡

  2. 软件环境：

  • 操作系统：Windows XP、 Vista、 Windows 7

  • IE浏览器：IE6.0以上，推荐IE8.0

  • 推荐显示器分辨率：1027*768及以上
