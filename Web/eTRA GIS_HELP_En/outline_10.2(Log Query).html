Log Query

The steps of querying the logs are as follows:

1) Click [Dispatch/Operation log] on the menu bar. The Query dialog box pops up, shown as below.

Log query

2) The dispatch user can select the filter criteria and then click <Search>. It will list the related operation logs.

Note:

  • The "start time" and "end time" are required, otherwise it will fail to search the data.

  • "The type of operation" is configured by the configuration user.
