Modify Mobile User

The steps of modifying the mobile user are as follows:

1) Click [Basis/Mobile user]. The Mobile User List pops up.

2) Click <Modify> in the right of the mobile user, and the “Modify” dialog box pops up. Modify the mobile user information based on actual needs, shown as below.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Modify the mobile user information

Note:

  • The dispatcher can modify the mobile user information of his unit and sub-units.

  • The modified terminal ID cannot repeat with other user’s terminal ID.

  • If the terminal ID does not exist, then the modification will fail.
