Half-Duplex Private Call

  • Call Toolbar: Initiate Half-Duplex Private Call; Receive Half-Duplex Private Call

  • Call Panel: Initiate Half-Duplex Private Call; Receive Half-Duplex Private Call

  • Uplink Call Panel: Receive Half-Duplex Private Call; Initiate Half-Duplex Private Call

The steps of initiating a half-duplex private call through call toolbar are as follows:

1) Select the user on the map, such as user 24004.

2) Click the Private Call icon on the call toolbar, and a menu bar appears on the right. Select “PTT mode”, shown in Figure 1.1.

Figure 1.1. Select the PTT mode

3) A Microphone icon will pop up beside the call toolbar, and an icon will appear on the side of the user on the map, and the status bar will display "(24004): The subscriber is ringing", shown in Figure 1.2.

Figure 1.2. The subscriber is ringing

4) After answering the call, the user can hold down the PTT button to speak, and the status bar will display "(24004): Obtain authorization", shown in Figure 1.1. When the user releases the PTT button, the status bar will display "(24004): PTT released".

Figure 1.1. Mobile user obtains permission

5) After the user releases the PTT button, the dispatcher can hold down the Microphone icon to speak, shown in Figure 1.2. Release the Microphone icon to release the permission.

Figure 1.2. Dispatcher obtains permission

6) Click the Private Call icon to end the call.

The steps of receiving a half-duplex private call through call toolbar are as follows:

1) If the dispatch console receives a half-duplex private call from a user, such as user 24004, an icon will appear on the side of the user on the map, and the status bar will display "(24004): Call access request", shown in Figure 1.3.

Figure 1.3. Call access request

2) Click the Private Call icon on the call toolbar, and a menu bar appears on the right. Select “Answer”. The call is set up, and the Microphone icon will pop up beside the call toolbar.

3) The dispatcher can hold down the Microphone icon to speak. After the dispatcher releases the Microphone icon, the user can hold down the PTT button to speak.

4) Click the Private Call icon to end the call.

The steps of initiating a half-duplex private call through call panel are as follows:

1) Click [Dispatch/Call panel] on the menu bar and click "Private call”. The “Private call” dialog box pops up.

2) Enter the terminal ID, such as 24004. The system will get the corresponding information automatically.

3) Select the “Call mode” as “PTT mode”, and click <PTT>, shown in Figure 2.1.

Figure 2.1. The subscriber is ringing

4) After answering the call, the user can hold down the PTT button to speak, shown in Figure 2.2.

Figure 2.2. Mobile user obtains permission

5) After the user releases the PTT button, the dispatcher can press <PTT> to speak.

6) Click <Hang up> to end the call.

The steps of receiving a half-duplex private call through call panel are as follows:

1) If the dispatch console receives a half-duplex private call from the user 24004, the “Private call” dialog box is shown in Figure 2.3.

Figure 2.3. Call access request

2) Click <PTT>, and then press <PTT> to speak, shown in Figure 2.4.

Figure 2.4. Dispatcher obtains permission

3) After the dispatcher releases <PTT>, the user can hold down the PTT button to speak.

4) Click <Hang up> to end the call.

The steps of receiving a half-duplex private call through uplink call panel are as follows:

1) When the dispatch console receives a half-duplex private call from a user, such as user 24004, the private call panel is shown in Figure 3.1.

Figure 3.1. Call access request

2) Click <Answer>, and then press <PTT> to speak, shown in Figure 3.2.

Figure 3.2. Dispatcher obtains permission

3) After the dispatcher releases <PTT>, the user can hold down the PTT button to speak.

4) Click <Hang up> to end the call.

The steps of initiating a half-duplex private call through uplink call panel are as follows:

1) Click <Call> in the private call panel to initiate the call, shown in Figure 3.3.

Figure 3.3. The subscriber is ringing

2) After answering the call, the user can hold down the PTT button to speak, shown in Figure 3.4.

Figure 3.4. Mobile user obtains permission

3) After the user releases the PTT button, the dispatcher can press <PTT> to speak.

4) Click <Hang up> to end the call.

Note:

In the uplink call panel, you can only initiate a half-duplex private call to the terminal that already has the half-duplex private call records.
