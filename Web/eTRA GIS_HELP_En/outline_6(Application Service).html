Application Service

There are several methods to perform the application service:

1) Right-click a mobile user on the map and select the application service, shown in Figure 1.

Figure 1. Right-click the user and select the application service

2) Click [Dispatch/Call panel] on the menu bar and select the application service on the call panel, shown in Figure 2.

Figue 2. Select the application service on the call panel

3) Select a region of users on the map and select the application service in the user list, shown in Figure 3.

Figure 3. Select the application service through region selection

4) Open the member tree and select the application service in the user list, shown in Figure 4.

Figure 4. Select the application service through member tree

5) Right-click on the map and select [Search police], and select the application service in the search list, shown in Figure 5.

Figure 5. Search the user and select the application service

The application services include:

Lock

Watchlist

Electronic Fence

Real-Time Display

Real-Time Trajectory

Historical Trajectory

Rapid Positioning

User Details
