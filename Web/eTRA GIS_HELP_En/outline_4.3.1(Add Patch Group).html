Add Patch Group

The steps of adding the patch group are as follows:

1) Click [Marshaling/Patch group]. The Patch Group List pops up, shown in Figure 1.

Figure 1. Patch Group List

2) Click <Add>, and the “Add” dialog box pops up. Enter the patch group information in the dialog box, shown in Figure 2.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Figure 2. Add the patch group information

Note:

  • The system allows up to eight patch groups to be created.

  • A patch group should contain at least one group. A group can only be added to one patch group.

  • The names of the patch groups must be different in a same unit.
