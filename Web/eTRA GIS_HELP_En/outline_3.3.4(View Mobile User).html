View Mobile User

The steps of viewing the mobile user are as follows:

1) Click [Basis/Mobile user]. The Mobile User List pops up.

2) Select the unit, or enter the user name/number/terminal ID. Click <Search>. The list will display the mobile user information, shown in Figure 1.

Figure 1. Query the Mobile User List

3) Click the mobile user name to display the details, shown in Figure 2.

Figure 2. View the mobile user information

Note:

  • The dispatcher can view the mobile user information of his unit and sub-units.

  • If the mobile user does not exist, the system will prompt, "Can not find the relevant data".
