Delete Terminal

The steps of deleting the terminal are as follows:

1) Click [Basis/Terminal]. The Terminal List pops up.

2) Click <Delete> in the right of the terminal. A confirmation window pops up.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Note:

  • If a terminal is in a call, then it cannot be deleted.

  • If there is a patch group or multi-select group associated with the terminal, then it cannot be deleted.

  • If the terminal and mobile user are in bound state, unbind the relations first.
