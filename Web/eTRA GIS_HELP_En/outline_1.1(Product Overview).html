Product Overview

eTRA GIS is the geographic information subsystem of ETRA (Eastcom Trunked Radio) system developed by Eastern communications Co., Ltd. It mainly serves for the private network customers with terrestrial wireless dispatching needs, such as Public Security Bureau, Fire Department, Armed Police and Government Sector.


eTRA GIS integrates the wireless dispatching communications, GIS (Geographic Information System) and GPS (Global Positioning System) effectively. It is featured with a visualized, efficient, convenient and brand-new dispatching mode-dispatching by clicking and operating on the map.


eTRA GIS is a multimedia service terminal. It enables the dispatchers to implement dispatching control and service management on mobile stations, fixed radios and mobile radios. It has various digital trunking dispatching functions, such as voice dispatching, messaging and GIS application, as well as supports the interface to the Public Security GIS Platform.


The system is connected to the WEB server via IP network, to access to the GPS information of end-users and display the information on the map. Meanwhile, it supports hierarchical dispatching. Dispatchers of different levels have different operating permissions.

eTRA GIS system architecture

eTRA GIS is composed of the GPS resolution server, WEB GIS server, and dispatch client.

  1. The functions of the GPS resolution server are as follows:

  • GPS position resolution

  • GPS data storage

  • Supports interface to a third-party GIS platform

  1. The functions of the WEB GIS server are as follows:

  • Provides GIS service

  • Provides WEB service

  • User configuration management

  • Map management

  • Core database management

  1. The functions of the GIS dispatch console are as follows:

  • Provides monitoring and dispatching function

  • An operator terminal for commanding and dispatching

  • Users can equip one or more dispatch consoles based on the operating requirements.
