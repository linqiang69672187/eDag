Lock

We can lock a mobile user. Once in locked mode, the user will be marked with a special icon, and the map will make automatic adjustments constantly to make sure that the user is always in the visible region.

Here are some common methods of locking/unlocking the user:

1) Right-click a mobile user on the map and select [Application service/Lock]. A lock icon will appear in the upper right corner of the user, shown in Figure 1. When the user leaves the current view, the center of the map will immediately navigate to the location of the user. Right-click the user and select [Application service/Unlock]. The lock icon will disappear.

Figure 1. Lock the user

2) Alternatively, click [Dispatch/Call panel] on the menu bar, and click "Lock function”. The “Lock” dialog box pops up, shown in Figure 2.

Figure 2. "Lock” dialog box

3) Enter the terminal ID, or click <Select> to select the user, shown in Figure 3.

Figure 3. Select the user

4) The dialog box will display the lock status of the user, shown in Figure 4. If the user is unlocked, click <Lock> to lock the user, and if the user is locked before, click <Unlock> and the user will be unlocked.

Figure 4. The lock status of the user

Note:

  • Only one mobile user can be locked. If you want to lock another user, unlock the previous one.

  • After locking the mobile user, we cannot zoom the map in or out.

  • If the latitude and longitude of a mobile user is 0 or does not exist, then the user cannot be locked.
