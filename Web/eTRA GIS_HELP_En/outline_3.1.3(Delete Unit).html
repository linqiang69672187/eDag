Delete Unit

Only the first-level dispatcher can delete the unit information.

The steps of deleting the unit are as follows:

1) Click [Basis/Unit]. The Unit List pops up.

2) Click <Delete> in the right of the unit. A confirmation window pops up.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Note:

  • If the unit is a first-level unit, or has subordinate units, or the dispatcher belongs to it, or it is the higher-level unit of the dispatcherâ€™s unit, it will fail to delete the unit.

  • If a unit has the associated data table (including the records of real-time GIS/historical GIS/mobile user/dispatcher/terminal/grouping information), it will fail to delete the unit.

  • A unit can be deleted successfully only by deleting all the GIS information associated to it.
