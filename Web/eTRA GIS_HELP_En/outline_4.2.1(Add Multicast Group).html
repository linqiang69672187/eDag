Add Multicast Group

The steps of adding the multicast group are as follows:

1) Click [Marshaling/Multicast group]. The Multicast Group List pops up, shown in Figure 1.

Figure 1. Multicast Group List

2) Click <Add>, and the “Add” dialog box pops up. Enter the multicast group information in the dialog box, shown in Figure 2.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Figure 2. Add the multicast group information

Note:

  • The multicast group ID is unique in the system. If the entered ID that already exists, it will fail to add the multicast group.

  • A multicast group should contain at least one group. A group can only be added to one multicast group.
