Add Dispatch Console

The steps of adding the dispatch console are as follows:

1) Click [Basis/Dispatch console]. The Dispatch Console List pops up, shown in Figure 1.

Figure 1. Dispatch Console List

2) Click <Add>, and the “Add” dialog box pops up. Enter the dispatch console information in the dialog box, shown in Figure 2.

3) Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.

Figure 2. Add the dispatch console information

Note:

  • The dispatch console ID is unique in the system. If the entered ID already exists, it will fail to add the dispatch console.

  • The dispatcher could only add the dispatch console information of his unit and sub-units, instead of the higher-level units or equivalent units.
