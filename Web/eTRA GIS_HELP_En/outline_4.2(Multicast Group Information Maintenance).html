Multicast Group Information Maintenance

The multicast group information parametrs are as follows:

Name Description  Type    Default

Multicast group name

The name of the multicast group

Character, 1~10 string

N/A

Multicast group ID

The ID of the multicast group

Integer, 1~13999999

N/A

The unit

To which unit the multicast group belongs

Character, 1~10 string

N/A

Member group

The groups associated to the multicast group

Can select multiple groups, and the multicast group contains the ID number of each group

 

 

This chapter mainly describes:

  • Add Multicast Group

  • Modify Multicast Group

  • Delete Multicast Group

  • View Multicast Group

Note:

When the multicast group information is created successfully, authorized dispatcher can initiate the “Multicast group call”.
