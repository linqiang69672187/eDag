Status Message

  • Send Status Message

  • Receive Status Message

The steps of sending status message are as follows:

1) Right-click a mobile user on the map and select [Application service/Status message]. The “Send status message” dialog box pops up, shown in Figure 1. Choose whether to add other users or groups, and select send as text or status number. Then click <Send>.

Figure 1. Send status message

2) Alternatively, click [Dispatch/Call panel] on the menu bar and click "Status message". The “Send status message” dialog box pops up, shown in Figure 2. Click <Select> to select the user, and select send as text or status number. Then click <Send>.

Figure 2. “Send status message” dialog box
