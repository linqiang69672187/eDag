Add User Type

The steps of adding the user type are as follows:

1) Click [Basis/User type]. The User Type List pops up, shown in Figure 1.

Figure 1. User Type List

2) Click <Add>. The “Add” dialog box pops up, shown in Figure 2.

Figure 2. Add the user type

3) Enter the type name, and click <Select icon>. The “Select the user type icon” dialog box pops up, shown in Figure 3. We can choose a set of icons as the status icons of the user type.

Figure 3. Select the user type icon group

4) Alternatively, click <Add> in the bottom of the dialog box. The “New user type icon group” dialog box pops up, shown in Figure 4.

Figure 4. New user type icon group

5) Add the name of the icon group based on actual needs, and click <Browse> to upload the corresponding status icons.

6) Click <OK>, and the “Succeeded to add” dialog box pops up. The corresponding icons will appear in the “Add” dialog box. Click <OK>. If succeeded, the system will prompt the success; if failed, it will prompt the cause of error.
