Real-Time Display

If the real-time display function is enabled, when the system receives the position information of a mobile user, it will display the user’s icon and corresponding information on the screen. If the real-time display function is disabled, then the user’s icon and information will be hidden.

Here are some common methods of enabling/disabling the real-time display:

1) Right-click a mobile user on the map and select [Application service/Close display], shown in Figure 1. The user will disappear on the map.

Figure 1. Close the display

2) If you want to redisplay the user, search the user and check the “Display” column, shown in Figure 2.

Figure 2. Open the display

3) Alternatively, click [Dispatch/Call panel] on the menu bar, and click "Real-time display”. The “Real-time display” dialog box pops up, shown in Figure 3.

Figure 3. "Real-time display" dialog box

4) Enter the terminal ID, or click <Select> to select the user. The dialog box will show the display status of the user, shown in Figure 4.

Figure 4. The display status of the user

5) If the status is “Displaying”, click <OK> to hide the user. If the status is “Hiding”, click <OK> to display the user.
