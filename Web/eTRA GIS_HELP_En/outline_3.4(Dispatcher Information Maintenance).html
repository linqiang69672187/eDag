Dispatcher Information Maintenance

The dispatcher name and ID is unique in the system. When we create a first-level unit, the system will automatically generate a first-level dispatcher. The username is admin and the password is admin.

The dispatcher information parametrs are as follows:

Name Description  Type    Default

The unit

To which unit the dispatcher belongs

Character, 1~10 string

N/A

Username

The username of the dispatcher

4~20 characters of letters, figures or underscores

N/A

Password

The password used to log in to the GIS system

Character, 1~13 string

N/A
