Full-Duplex Private Call

  • Call Toolbar: Initiate Full-Duplex Private Call; Receive Full-Duplex Private Call

  • Call Panel: Initiate Full-Duplex Private Call; Receive Full-Duplex Private Call

  • Uplink Call Panel: Receive Full-Duplex Private Call; Initiate Full-Duplex Private Call

The steps of initiating a full-duplex private call through call toolbar are as follows:

1) Select the user on the map, such as user 24004.

2) Click the Private Call icon on the call toolbar, and a menu bar appears on the right. Select “Tel mode”, shown in Figure 1.1.

Figure 1.1. Select the Tel mode

3) An icon will appear on the side of the user on the map, and the status bar will display "(24004): The subscriber is ringing", shown in Figure 1.2.

Figure 1.2. The subscriber is ringing

4) After the user picks up, the call is set up. The status bar will display "(24004): The subscriber has answered", shown in Figure 1.3.

Figure 1.3. The subscriber has answered

5) Click the Private Call icon to end the call.

The steps of receiving a full-duplex private call through call toolbar are as follows:

1) If the dispatch console receives a full-duplex private call from a user, such as user 24004, an icon will appear on the side of the user on the map, and the status bar will display "(24004): Call access request", shown in Figure 1.4.

Figure 1.4. Call access request

2) Click the Private Call icon on the call toolbar, and a menu bar appears on the right. Select “Answer”, shown in Figure 1.5.

Figure 1.5. Answer the call

3) Then the call is set up, and the status bar will display "(24004): Call is set up", shown in Figure 1.6.

Figure 1.6. Call is set up

4) Click the Private Call icon to end the call.

The steps of initiating a full-duplex private call through call panel are as follows:

1) Click [Dispatch/Call panel] on the menu bar and click "Private call”. The “Private call” dialog box pops up.

2) Enter the terminal ID, such as 24004. The system will get the corresponding information automatically.

3) Select the “Call mode” as “Tel mode”, and click <Call>, shown in Figure 2.1.

Figure 2.1. The subscriber is ringing

4) After the user picks up, the call begins, shown in Figure 2.2.

Figure 2.2. The subscriber has answered

5) Click <Hang up> to end the call.

The steps of receiving a full-duplex private call through call panel are as follows:

1) If the dispatch console receives a full-duplex private call from the user 24004, the “Private call” dialog box is shown in Figure 2.4.

Figure 2.4. Call access request

2) Click <Answer>, and the call is set up, shown in Figure 2.5.

Figure 2.5. Call is set up

3) Click <Hang up> to end the call.

 

The steps of receiving a full-duplex private call through uplink call panel are as follows:

1) When the dispatch console receives a full-duplex private call from a user, such as user 24004, the private call panel is shown in Figure 3.1.

Figure 3.1. Call access request

2) Click <Answer>, and the call is set up, shown in Figure 3.2.

Figure 3.2. Call is set up

3) Click <Hang up> to end the call.

The steps of initiating a full-duplex private call through uplink call panel are as follows:

1) Click <Call> in the private call panel to initiate the call, shown in Figure 3.5.

Figure 3.5. The subscriber is ringing

2) After the user picks up, the call is set up, shown in Figure 3.6.

Figure 3.6. The subscriber has answered

3) Click <Hang up> to end the call.

Note:

  • In the uplink call panel, you can only initiate a full-duplex private call to the terminal that already has the full-duplex private call records.

  • If you initiate a call to a mobile user who has not registered in the system, the window will prompt, “The subscriber hangs up”.
