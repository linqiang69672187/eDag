GPS Report Parameter Setting

The steps of setting the GPS report parameters are as follows:

1) Click [Dispatch/Call panel] on the menu bar and click "GPS Report Parameter setting”. The “GPS Report Parameter setting” dialog box pops up, shown in Figure 1.

Figure 1. "GPS Report Parameter setting” dialog box

2) Specify the users. Enter the terminal ID in the dialog box and click <Add>; or click <Select> to select the users. Then set the GPS report period and destination based on actual needs, shown in Figure 2.

Figure 2. Set the GPS report parameters

3) Click <OK>, and the prompt box pops up.

4) The log window will display the instruction execution result, shown in Figure 3.

Figure 3. Instruction execution result
