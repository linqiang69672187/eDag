Member Tree

We can perform the following operations in the member tree:

  • Implement Services

  • View User Information

  • Online User Statistics

  • Quick Search

Implement Services

Open the member tree, and select the User/Group/Dispatch. Click a unit, and it will display all users in the unit. Click the Operate key to select the corresponding service, shown in Figure 1.

Figure 1. Implement services

View User Information

The member tree is designed in accordance with the tree structure of the administrative system. It displays the affiliation of each unit visually.

For example, view the information of a user. Click the unit to which the user belongs in the member tree. It will display all users in the unit. Then click the user name, and the user information dialog box pops up, shown in Figure 2.

Figure 2. View the user information

Online User Statistics

The member can display the number of online users. For example, " Hangzhou Public Security Bureau [10/20]", it means that the unit has a total of 20 people, and currently it has 10 online people.

Quick Search

There is a search box in the user list. Dispatcher can enter the name/number/ISSI, and click the Search button to search all eligible users in the system quickly, shown in Figure 3.

Figure 3. Quick search
