Historical Trajectory

If the historical trajectory function is enabled, the system can replay the trajectories of a user in a certain period.

Here are some common methods of enabling/disabling the historical trajectory:

1) Right-click a mobile user on the map and select [Application service/Historical trajectory]. The user's historical trajectory configuration dialog box pops up. Configure the time period and trajectory style, shown in Figure 1.

Figure 1. View the historical trajectory

2) Click <OK>. The system will re-open an IE page, which displays the trajectories of the user in this period. Close the browser to end the playback.

3) Alternatively, click [Dispatch/Call panel] on the menu bar, and click "Historical trajectory”. The “View historical trajectory” dialog box pops up. Enter the terminal ID, or click <Select> to select the user, and then configure the time period and trajectory style, shown in Figure 2.

Figure 2. View the historical trajectory

4) Click <OK>. The system will re-open an IE page, which displays the trajectories of the user in this period. Close the browser to end the playback.

Note:

  • The maximum time span of a single query is 24 hours.

  • If the data is empty in the specified period, the system will prompt, “No historical trajectory record”.
