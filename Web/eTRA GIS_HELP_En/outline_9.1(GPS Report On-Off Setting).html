GPS Report On-Off Setting

The steps of GPS report on-off setting are as follows:

1) Click "" in the map toolbar and select a region of one or more mobile users. The user list pops up, and click the <GPS Report On-Off> button.

2) Alternatively, click [Dispatch/Call panel] on the menu bar and click "GPS Report On-Off”. The “GPS Report On-Off” dialog box pops up.

3) Enter the terminal ID in the dialog box and click <Add>; or click <Select> to select the users, shown in Figure 1.

Figure 1. Specify the users

4) After specifying the users, you can click <Close> or <Open> in the dialog box. If you click <Close>, then the selected users will not report the GPS information to the system, otherwise click <Open>.

5) Then a prompt box pops up, shown in Figure 2.

Figure 2. The instruction prompt box

6) Click <OK>. The log window will display the instruction execution result, shown in Figure 3.

Figure 3. Instruction execution result
