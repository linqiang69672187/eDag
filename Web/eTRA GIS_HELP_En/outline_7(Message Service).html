Message Service

There are several methods to perform the message service:

1) Right-click a mobile user on the map and select the message service, shown in Figure 1.

Figure 1. Right-click the user and select the message service

2) Click [Dispatch/Call panel] on the menu bar and select the message service on the call panel, shown in Figure 2.

Figure 2. Select the message service on the call panel

3) Select a region of users on the map and select the message service in the user list, shown in Figure 3, or click the <Send SMS> button for mass messaging.

Figure 3. Select the message service through region selection

4) Open the member tree and select the message service in the user list, shown in Figure 4.

Figure 4. Select the message service through member tree

5) Right-click on the map and select [Search police], and select the message service in the search list, shown in Figure 5.

Figure 5. Search the user and select the message service

The message services include:

Private SMS

Group SMS

Bulk SMS

Status Message
