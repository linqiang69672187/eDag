Search Mobile User

The steps of searching the mobile user are as follows:

1) Right-click anywhere on the map and select [Search police]. The Search dialog box pops up, shown in Figure 1.

Figure 1. Search dialog box

2) Enter the name/number/ISSI, and click <Search>. The dialog box will display the mobile user information, shown in Figure 2.

Figure 2. Search the mobile user

3) You can display/hide/locate the users, and perform dispatch/SMS/application services. You can also click the name of the user to view the user details, shown in Figure 3.

Figure 3. User operation
